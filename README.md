# KrestikiNoliki

## Крестики-Нолики для Android 

Простое приложение "Крестики-Нолики" для Android, созданное на Java.

## Особенности

- Два игрока (X и O) по очереди делают ходы
- Отслеживание счета для обоих игроков
- Кнопка "Сброс" для начала новой игры
- Минималистичный и интуитивно понятный интерфейс

## Как играть

1. Игроки по очереди нажимают на клетки поля
2. Игрок A всегда играет крестиками (X), Игрок B - ноликами (O)
3. Первый игрок, собравший линию из трех своих символов по горизонтали, вертикали или диагонали, побеждает
4. Для начала новой игры нажмите кнопку "Сброс"

## Установка

1. Скачайте APK-файл из раздела 
2. Установите приложение на свое Android-устройство
3. Если необходимо, разрешите установку из неизвестных источников в настройках устройства

## Технологии

- Java
- Android Studio
- XML для разметки интерфейса
